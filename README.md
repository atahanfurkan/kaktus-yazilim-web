Kaktüs Yazılım Web Sitesi
Kaktüs Yazılım için React ve Vite kullanarak hazırladığım kurumsal ana sayfa projesi.

🌐 Canlı Link
Projeyi buradan inceleyebilirsiniz: kaktusyazilim.netlify.app

Neler Yaptım?
React bileşen yapısını kullanarak sayfayı modüllere ayırdım.
Sayfanın mobilde ve masaüstünde düzgün görünmesi için responsive (uyumlu) tasarım yaptım.
Vite kullanarak hızlı bir geliştirme ortamı oluşturdum.

Kullandığım Teknolojiler
React.js
Vite
CSS3

Bilgisayarınızda Çalıştırmak İçin
git clone https://github.com/atahanfurkan/kaktus-yazilim-web.git
npm install
npm run dev
